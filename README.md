### Clone to local

I recommend the use of [@antfu/ni](https://github.com/antfu/ni) for this project

```sh
$ npx degit NicolaSpadari/yt-clone my-app
$ cd my-app
$ ni
```

### Development

Run and visit http://localhost:3000

```sh
$ nr dev
```

### Linting

To build, run

```sh
$ nr lint
```

### Build

To build, run

```sh
$ nr build
```

This will create the `dist` folder ready to be uploaded
